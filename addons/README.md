# Duonordic Nextjs Addons

## Add-on Module Naming

Names for modules should be direct and
descriptive of their function, often with a premium or elevated feel.

- Insights: For a user dashboard or analytics module.
- Syndicate: For a blog or content management module.
- Summit: For a high-end reporting or analytics module.
- Forge: For a powerful editor or customization tool.
- Pinnacle: For an SEO or performance optimization module.
- Flow: For an e-commerce or conversion-focused module.
