# test2 Template

A basic-level template for the Duonordic platform.

## Package Structure

This template is a standalone package that can be imported as:
```typescript
import { TEST2_TEMPLATE } from "@dn/templates/basic/test2";
// or
import { TEST2_TEMPLATE } from "@dn/templates/basic";
```

## Features

- Modern design
- Responsive layout
- SEO optimized
- Easy to customize

## Development

```bash
cd packages/templates/basic/test2
pnpm dev    # Watch mode
pnpm build  # Build package
pnpm lint   # Lint code
```
